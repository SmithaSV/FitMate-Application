# FitMate

FitMate is a full-stack fitness and habit tracking application that helps users manage fitness goals and log daily activities.

---

## Features
- User registration and login
- JWT-based authentication
- Create and manage fitness goals
- Log daily activities
- Secure logout using token invalidation
- Improved performance using Redis caching

---

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- SQL Server
- Redis
- Maven

### Frontend
- React
- JavaScript
- HTML, CSS
- Axios (API communication)
